# Reconnaissance

The two phase step of the penetration testing process:

1. Passive information gathering / OSINT (open-source intelligence)
   * Web information
   * Email harvesting
   * Whois enumeration
2. Active information gathering - **illegal without permission**
   * OS fingerprinting
   * Port scanning
   * DNS enumeration
